# min-width

[min-width](https://developer.mozilla.org/en-US/docs/Web/CSS/min-width) 속성에 대한 유틸리티입니다.

요소의 최소 너비를 설정합니다. 이는 <code>width</code>의 값이 <code>min-with</code>로 지정된 값보다 작아지지 않도록 합니다.
