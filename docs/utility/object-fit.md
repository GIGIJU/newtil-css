# object-fit

[object-fit](https://developer.mozilla.org/en-US/docs/Web/CSS/object-fit) 속성에 대한 유틸리티입니다.

<code>img</code> 또는 <code>video</code>와 같은 대체 요소의 내용이 컨테이너에 맞게 크기를 조정하는 방법을 설정합니다.

  </thead>
