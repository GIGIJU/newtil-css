# font-weight

[font-weight](https://developer.mozilla.org/en-US/docs/Web/CSS/font-weight) 속성에 대한 유틸리티입니다.

폰트의 두께를 설정합니다. 사용 가능한 두께는 현재 설정된 <code>font-family</code>에 따라 달라집니다.
