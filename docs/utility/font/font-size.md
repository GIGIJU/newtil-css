# font-size

[font-size](https://developer.mozilla.org/en-US/docs/Web/CSS/font-size) 속성에 대한 유틸리티입니다.

폰트의 크기를 설정합니다. 폰트 크기를 변경하면 <code>em</code>, <code>ex</code>와 같은 폰트 크기 상대적인 <code>length</code> 단위의 크기도 업데이트됩니다.
