# overflow-x

[overflow-x](https://developer.mozilla.org/en-US/docs/Web/CSS/overflow-x) 속성에 대한 유틸리티입니다.

컨텐트가 블록 레벨 요소의 좌, 우 모서리를 넘어갈 때 어떤 내용을 표시할지를 설정합니다. 아무것도 표시하지 않을 수도, 스크롤바를 표시할 수도, <code>overflow</code>된 컨텐트가 표시될 수도 있습니다.
